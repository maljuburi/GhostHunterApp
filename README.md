# Hybrid Mobile App
### The application is coded using:
* HTML5
* CSS3
* Javascript


# Built with Cordova API
### Cordova Plugins used:
* Camera
* Device Orientation (Compass)
* Flash Light (Torch)


# The idea of the app is to hunt a ghost for the Halloween
